# Flow Roadmap

> A living document outlining the future direction of Flow.

---

## v0.1.x - Foundation

- [x] Core CLI commands (note, status, resume, done)
- [x] Git-aware context detection
- [x] JSON-based storage
- [x] Pre-commit hooks with prek

## v0.2.x - History & Insights

- [x] Context history tracking
- [x] History command
- [ ] Duration tracking
- [ ] Search/filter history

## v0.3.x - Enhanced Context

- [ ] Project-specific contexts
- [ ] Tags and categories
- [ ] Context templates
- [ ] Multi-line notes

## v0.4.x - Integrations

- [ ] IDE extensions (VS Code, etc.)
- [ ] Shell integration (prompt status)
- [ ] Git hooks integration
- [ ] Export/import (Markdown, JSON)

## v1.0.0 - Production Ready

- [ ] Stable API
- [ ] Comprehensive documentation
- [ ] Performance benchmarks
- [ ] Cross-platform testing

---

## Ideas & Experiments

Ideas that might or might not make it into a release:

- Cloud sync (optional)
- Team context sharing
- AI-powered context summaries
- Time tracking analytics
- Custom themes/output formats
